randfilter
===========

randfilter is tested with Python 2.6, 2.7, 3.2, 3.3.

Requirements
============

- Python 2.6 / 2.7 / 3.2 / 3.3 or higher

- docopt==0.6.1
- schema==0.2.0

Installation
============

Install from pypi
-----------------
::

    easy_install randfilter

or

::

    pip install randfilter

Install from github
-------------------
::

    git clone https://github.com/ton1517/randfilter.git
    cd randfilter
    python setup.py install
