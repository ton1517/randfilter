randfilter
===========

Requirements
============

- Python 2.6 / 2.7 / 3.2 / 3.3 or higher

I'm testing with 2.6.8, 2.7.5, 3.2.5, 3.3.2.

Installation
============

Install from pypi
-----------------
::

    easy_install randfilter

or

::

    pip install randfilter

Install from github
-------------------
::

    git clone https://github.com/ton1517/randfilter.git
    cd randfilter
    python setup.py install
